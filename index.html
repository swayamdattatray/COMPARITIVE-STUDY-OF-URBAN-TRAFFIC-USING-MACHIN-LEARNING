Project Overview

This project predicts traffic_flow using historical traffic and contextual attributes such as speed, time, day, and weather. It compares statistical, machine learning, and deep learning approaches.

  • Data preprocessing with missing value handling and scaling
  • Feature engineering using lag and rolling statistics
  • Train/test split (80:20) in chronological order

ARIMA

Classical time-series forecasting baseline for univariate traffic signal modeling.

Random Forest

Ensemble tree regressor for robust nonlinear relationships in engineered features.

XGBoost

Boosted tree model with strong tabular-data performance for traffic prediction.

LSTM

Sequence learning model for capturing temporal dependencies in traffic flow patterns.
